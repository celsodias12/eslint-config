# Installation

Install the package and its dependencies:

```bash
npm i -D @celsodias12/eslint-config-a11y eslint-plugin-jsx-a11y
```

## Configuration

In your ESLint configuration file, add the following content:

```json
{
  "extends": [
    // ... other configs
    "@celsodias12/eslint-config-a11y"
  ]
}
```
